# 一级标题

段落由空行分隔。

第二段。_斜体_、**粗体** 和 `等宽字体`。项目列表
看起来是这样的：

- 这一个
- 那一个
- 另外那一个

注意，不考虑星号，实际的文本
内容从第 4 列开始。

> 块引用
> 就是这样写的。
>
> 如果你愿意，
> 它们可以跨越多个段落。

使用 3 个破折号表示长划线。使用 2 个破折号表示范围（例如，"都在第 12--14 章"）。三个点 ... 将被转换为省略号。
支持 Unicode。☺

## 二级标题

这是一个编号列表：

1. 第一项
2. 第二项
3. 第三项

再次注意实际文本从第 4 列开始（距离左边 4 个字符）。这是一个代码示例：

    # 让我重申一下...
    for i in 1 .. 10 { do-something(i) }

正如你可能猜到的，缩进了 4 个空格。顺便说一下，你可以使用分隔块，而不是缩进块，如果你愿意：

```
define foobar() {
    print "欢迎来到风味之国!";
}
```

（这使得复制和粘贴更容易）。你可以选择标记分隔块让 Pandoc 对其进行语法高亮：

```python
import time
# 快速，数到十！
for i in range(10):
    # （但不要太快）
    time.sleep(0.5)
    print i
```

3. 把所有东西倒进锅里，按照
    这个算法：

        找到木勺
        揭开锅盖
        搅拌
